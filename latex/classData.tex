\doxysubsection{Detailed Description}
Classe \mbox{\hyperlink{classData}{Data}} representando uma string que deve possuir as seguintes regras de formatação\+: 


\begin{DoxyItemize}
\item Formato DD-\/\+MM-\/\+AAAA;
\item DD como o dia, valendo de 01 a 31;
\item MM como o mes, valendo de 01 a 12;
\item AA como o ano, valendo de 2000 a 2100;
\item Hifens sao usados para separar dias, meses e anos;
\item Anos bissextos sao considerados. 
\end{DoxyItemize}
